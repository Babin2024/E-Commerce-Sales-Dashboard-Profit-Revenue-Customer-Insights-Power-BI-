# 🛒 E-Commerce Sales Dashboard

## 📌 Project Overview
This project presents an **interactive E-commerce Sales Dashboard** designed to analyze **sales performance, profit trends, and customer behavior**.  
It helps businesses make **data-driven decisions** by visualizing key metrics.

---

## 🎯 Objectives
- Analyze sales and profit performance  
- Identify top-performing states and customers  
- Understand category-wise sales distribution  
- Track monthly profit trends  
- Analyze payment methods  

---

## 📊 Key KPIs
- **Total Profit:** 26K  
- **Total Sales Amount:** 220K  
- **Total Quantity Sold:** 1706  
- **Average Order Value (AOV):** 61.19K  

---

## 📈 Dashboard Insights

### 🗺️ Sales by State
- Maharashtra generates the highest sales  
- Followed by Madhya Pradesh and Uttar Pradesh  

### 📅 Monthly Profit Analysis
- Profit fluctuates across months  
- Some months show negative profit (loss)  
- Peak performance observed in selected months  

### 💳 Payment Mode Analysis
- **COD (39%)** is most used  
- Followed by UPI (19%) and Credit Card (16%)  
- EMI and Debit Card have lower usage  

### 👤 Top Customers
- Highest contributing customers:
  - Harivansh  
  - Madhav  
  - Madan Mohan  

### 🛍️ Category-wise Sales
- Clothing: **47% (highest)**  
- Electronics: 32%  
- Furniture: 21%  

### 📦 Sub-Category Profit
- Highest profit:
  - Printers  
  - Bookcases  
- Lower profit:
  - Tables  
  - Accessories
